# GeoPunch 考勤系统 - 小程序页面原型

## 一、页面结构总览

```
├── pages/
│   ├── index          # 首页（打卡入口+今日状态）
│   ├── checkin        # 打卡页面
│   ├── records        # 打卡记录
│   ├── leave          # 请假申请
│   ├── makeup         # 补卡申请
│   ├── approval       # 审批列表（管理员可见）
│   ├── profile        # 个人中心
│   └── settings       # 设置
```

---

## 二、首页（index）

### 页面说明
员工打开小程序的第一个页面，显示今日打卡状态和快捷操作入口。

### 页面元素

```
┌─────────────────────────────────────┐
│  [头像] 张三 ▼                      │  ← 顶部栏，右上角下拉菜单
│  研发部 | 研发行政                   │  ← 部门+员工类型标签
├─────────────────────────────────────┤
│                                     │
│     ┌─────────────────────────┐    │
│     │     📅 2026年05月08日     │    │  ← 当前日期
│     │      星期五               │    │
│     └─────────────────────────┘    │
│                                     │
│  ┌─────────────────────────────┐   │
│  │                             │   │
│  │     [ 上班打卡 ]            │   │  ← 主打卡按钮（根据状态变色）
│  │     08:30 已打卡            │   │  ← 显示打卡时间
│  │                             │   │
│  └─────────────────────────────┘   │
│                                     │
│  ┌─────────────────────────────┐   │
│  │  上班    08:30    ✅ 正常    │   │  ← 打卡记录卡片
│  │  下班    18:30    ✅ 正常    │   │
│  └─────────────────────────────┘   │
│                                     │
│  本月出勤率: 95.5%  异常: 2次       │  ← 统计摘要
│                                     │
├─────────────────────────────────────┤
│  [首页]  [记录]  [请假]  [我的]      │  ← 底部Tab导航
└─────────────────────────────────────┘
```

### 打卡按钮状态

| 状态 | 按钮颜色 | 按钮文字 | 副文字 |
|------|---------|---------|--------|
| 未打卡 | 蓝色(主色) | 上班打卡 | 点击签到 |
| 已打卡(正常) | 绿色 | 已打卡 | 08:30 正常 |
| 已打卡(迟到) | 红色 | 已打卡 | 09:15 迟到 |
| 未到打卡时间 | 灰色 | 暂未开放 | 07:00后可打卡 |
| 超出范围 | 橙色 | 位置异常 | 请到公司范围内打卡 |

### 交互说明
1. 点击打卡按钮 → 调用GPS获取位置 → 验证是否在范围内 → 打卡成功/失败提示
2. 点击"查看详情" → 跳转至打卡记录页面
3. 底部Tab切换页面

## 三、打卡页面（checkin）

│  │     距离公司: 320米           │   │  ← 距离开启位置距离
│  └─────────────────────────────┘   │
│                                     │
│     ┌─────────────────────────┐    │
│     │                         │    │
│     │    🗺️  显示地图          │    │  ← 地图组件

│     │                         │    │
│     └─────────────────────────┘    │
│                                     │
│  ┌─────────────────────────────┐   │
│  │ 允许打卡范围: 500米           │   │
│  │ 当前状态: ✅ 在范围内         │   │
│  └─────────────────────────────┘   │
│                                     │
│  ┌─────────────────────────────┐   │

4. 拍照打卡开关：开启后打卡时调用相机拍照

## 十、关键交互流程

### 10.1 员工打卡流程
```
首页点击打卡按钮
    ↓
checkin页面获取GPS
    ↓
判断是否在允许范围内
    ├── 在范围内 → 显示确认打卡按钮 → 点击 → 打卡成功 → 返回首页
    └── 不在范围内 → 提示"不在打卡范围" → 返回首页
```

### 10.2 请假申请流程
```
我的页面点击"请假申请"
    ↓
leave页面填写表单
    ↓
提交申请
    ↓
系统创建待审批记录 → 通知部门管理员
    ↓
部门管理员审批
    ├── 通过 → 员工收到通知 → 请假记录更新为"已通过"
    └── 驳回 → 员工收到通知 → 请假记录更新为"已驳回"+意见
```

### 10.3 补卡申请流程（仅研发行政）
```
打卡记录页面 → 发现缺勤记录 → 点击"申请补卡"
    ↓
makeup页面填写表单
    ↓
提交申请
    ↓
系统校验: 补卡日期≤3日? 本月次数<3次?
    ├── 校验失败 → 提示错误
    └── 校验通过 → 创建待审批记录 → 通知部门管理员
    ↓
部门管理员审批
    ├── 通过 → 打卡记录更新为"补卡"状态
    └── 驳回 → 员工收到通知
```
